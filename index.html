Assignment 14 - Reading Server Side JSON

This assignment showcases the process of setting up a Node.js project with Express hosted on GitHub and Render. The task involves creating a server that serves JSON data representing an array of crafts. The server fetches this data from a JSON file hosted on a GitHub page and serves it to clients via a GET request. On the client side, an index.html page with accompanying script.js and styles.css is created.
