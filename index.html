CSCE 242 - Victor Nguyen

This is class CSCE242 where we learn how to do web development. We learn html/css and javascript to create web applications.

Assignments

Assignment 01 - Basic HTML

The assignment implements basic HTML as a website layout. We used elements such as title, paragraphs, headers, and created a table. We also sourced images locally.

Assignment 02 - Basic CSS

The assignment incorporates CSS styling onto an HTML site. The styling showcases a website with a custom color scheme and a site that is responsive to both mobile and desktop.

Assignment 03 - Page Layout

The assignment demonstrates the use of flexbox page layout.The flexbox styling allows to achieve a column style navbar layout. The webpage also showcases multiple displays of flex for the latter half of the page.

Assignment 04 - Recreate CSS Page Engineering

We recreated a site from the College of Arts and Sciences. The assignment demonstrates the use of flexbox page layout.

Assignment 07 - Intro to JavaScript

This is our intital useage of Javascript. We learn how to do simple javascript scripts.
